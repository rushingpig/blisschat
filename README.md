# BlissChat

> BlissChat是一个基于nodejs + socket.io搭建的简易版的聊天室系统

## Prepare
- [x] node.js  仅此而已
- [x] 浏览器(`chrome / firefox /safari etc`)


## Quick Start

```bash
npm install && node server.js

```
## Technology Stack
- [x] nodejs
- [x] socket.io
- [x] express
